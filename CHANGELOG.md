# Changelog

## v0.6.1

### 新增
- IPV6 支持
- 自定义隧道日志等级

### 修复
- 关闭多路复用开关导致无法连接到节点的问题
- 元素毛玻璃效果依然不生效的问题
- 根据节点UDP支持动态调整连接优化，避免开启连接优化导致的无法使用问题
- 避免隧道页面切页后，隧道启动没有反馈的问题
- 屏幕太小导致更新窗口页面不完整的问题
- 更新进度提示一直是0%的问题


## v0.5.7

### 新增
- 部分操作音效
- Frpc 代理功能，允许客户端与节点使用代理进行连接
- 多路复用功能及开关
- 强制 TLS 功能及开关
- 连接优化功能及开关
- 修改隧道后重启映射功能及开关

### 优化
- 优化日志格式，新增颜色区分
- 优化隧道页面的隧道样式，提升设计一致性
- 重构Frpc启动逻辑，优化隧道启动流程，显著提升隧道启动速度
- 移除隧道启动限制，支持同时启动多个隧道

### 修复
- 自定义隧道日志名称显示不正确的问题
- 元素毛玻璃效果不生效的问题
- 从其他页面回来后，启动过的隧道进度条会出现的问题
